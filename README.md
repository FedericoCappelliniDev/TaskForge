# TaskForge

TaskForge is a full-stack project and task management application built to showcase a modern Java + Angular architecture. The project includes secure JWT authentication, role-based authorization, project and task management, filtering, validation, and a 
responsive frontend powered by Angular.

# Dependencies
JDK 25.\
Node.js 22 LTS.\
Angular CLI 21.\
Docker Desktop\
PostgreSQL 16 or 17.
